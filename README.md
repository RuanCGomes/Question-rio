# 📝Pesquisa de Opinião

Este código foi feito para ser um questionário de opinião de um atendimento hipotético.
Pergunta se o atendimento foi excelente, bom ou ruim. Além disso, funciona em loop! E após 50 entrevistas, lista em quantas vezes ocorreu uma opinião positiva ou negativa.
Bem como os meus outros projetos, foi feito com código **Python**, dessa vez usando-o de verdade, para a atividade de DSI da Agenda 08.
